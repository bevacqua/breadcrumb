# Breadcrumb

> Follow breadcrumbs to a piece of cake

# Install

```shell
npm install breadcrumb --save
```

```shell
bower install breadcrumb.js --save
```

# `breadcrumb(cake, crumbs)`

Returns the value at `crumbs`. Example:

```js
breadcrumb({ foo: { bar: 'baz' } }, 'foo.bar');
// 'baz'
```
